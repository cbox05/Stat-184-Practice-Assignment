Busiest Airports Data Analysis

This project is a reproducible data analysis completed for STAT 184. The goal of the project is to analyze passenger traffic trends across major international airports over time and demonstrate a full workflow including data wrangling, visualization, and simulation techniques.

Data Source

The airport dataset was created using data from Wikipedia’s “List of busiest airports by passenger traffic.” The data includes yearly passenger totals for six major airports (ATL, DXB, FRA, HND, LHR, PKX) from 2020–2025.

The dataset was cleaned and transformed into a tidy format so that each row represents one airport in a given year, allowing for easier comparison and visualization.

Project Overview

This analysis focuses on:

Comparing passenger traffic across major airports
Identifying trends over time (including post-2020 recovery)
Visualizing differences using tables and line plots
Applying Monte Carlo simulation to estimate an integral

The project also includes a section on planning and evaluating GenAI outputs in a structured way.

Current Plan

The current state of the project includes:

Cleaned and structured dataset
Completed visualizations and tables
Monte Carlo simulation implemented
Final PDF generated using Quarto

Future improvements could include expanding the dataset or adding more detailed statistical analysis.
